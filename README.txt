CineVOX
Sua plataforma de filmes personalizada.

Introdução:
O CINEVOX é um projeto de gerenciamento e exibição de informações de filmes que permite aos usuários:
    • Explorar filmes populares e categorias específicas.
    • Visualizar trailers e detalhes completos sobre os filmes.
    • Comentar e interagir com outros usuários.
    • Adicionar filmes aos favoritos.

Este projeto foi desenvolvido com foco em uma experiência intuitiva e personalizada.

Ferramentas utilizadas
    • BootStrap
    • SQlite
    • PHP
    • HTML / CSS
    • JavaScript

Clone o repositório:
    git clone https://github.com/LeoX09/Projeto-CineVOX.git

    Uso:
Página principal
    Exibe os filmes populares, organizados em cards clicáveis.
    Pesquise filmes ao pressionar "Enter".

Categorias
    Acesse categorias específicas, organizadas horizontalmente.

Detalhes do Filme
    Pôster, trailer dublado, elenco, e informações detalhadas exibidos de forma responsiva.

Favoritos
    Adicione filmes à sua lista de favoritos para acesso rápido.

Sistema de Comentários
    Permite que usuários registrados deixem comentários em cada filme.

    Estrutura do Projeto:
Pastas principais

    css/: Arquivos CSS para estilização.
    img/: Imagens estáticas.
    node_modules/: Dependências do Node.js.
    database/: Banco de dados SQLite.

Arquivos principais

    index.php: Página inicial.
    info_filmes.php: Exibição de detalhes do filme.
    categorias.php: Exibição de filmes por categoria.
    registro.php, login.php, recuperacao.php: Gerenciamento de usuários.
